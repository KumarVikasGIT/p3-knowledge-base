# P3 Support

This is the public content repository for [help.p3.app](https://help.p3.app) — P3's documentation, help articles, news, and issue tracker.

P3 is a privacy-first, end-to-end encrypted photo delivery and sharing platform. This repo holds everything you need to understand how P3 works, get help when something's not working, and see what's actively being worked on.

## What's here

- **[`/docs`](./docs)** — Technical documentation. How P3's encryption model actually works, what data P3 can and can't see, and the reasoning behind key architectural decisions. Written for people who want the fuller picture.
- **[`/articles`](./articles)** — Help articles. Step-by-step guides for everyday tasks: creating galleries, managing your account, billing, recovering access if you're locked out, and troubleshooting.
- **[`/news`](./news)** — Product updates and release notes.
- **Issues** — Known bugs and feature requests, tracked publicly. [Browse open issues](../../issues) or [open a new one](../../issues/new/choose).

## Why this repo is public

P3 is built around the idea that the server shouldn't have to be trusted blindly — encryption that's actually verifiable, not just claimed. Keeping our documentation and issue tracker public is the same principle applied to how we communicate: nothing about how P3 works, or what's currently broken, is hidden from the people using it.

This repository contains only public-facing content. It is not the P3 product's source code and shares no infrastructure with it.
