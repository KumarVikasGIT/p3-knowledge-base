# P3 Support is now live

Starting today, [help.p3.app](https://help.p3.app) is the home for everything related to getting help with P3 — documentation explaining how the platform actually works, step-by-step help articles for common tasks, and a public issue tracker so you can see exactly what's being worked on.

## Why this is structured the way it is

Everything here is sourced from a public GitHub repository rather than a closed internal system. That's a deliberate choice: it means the documentation, the open issues, and the history of changes to both are all visible to anyone, not just to P3's support team. If you're curious how something works, or want to check whether a bug you've hit is already known, the answer is here rather than locked behind a support ticket.

## What's here today

- **[Documentation](/docs)** — the fuller, more technical explanations, starting with how P3's end-to-end encryption actually works
- **[Help articles](/help)** — practical guides for everyday tasks, from creating your first gallery to recovering access if you're ever locked out
- **[Issues](/issues)** — known bugs and feature requests, tracked in the open

If you don't have a GitHub account, every path here has an email fallback — see the [Contact](/contact) page.

This is a first version. If something's missing, confusing, or wrong, [tell us](/contact).
