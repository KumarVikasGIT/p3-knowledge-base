# How to create a gallery

From your dashboard, click **Create Gallery**.

You'll set:

- **Gallery name** — required
- **Description** — optional
- **Event date** — optional, useful for sorting if you manage many galleries
- **Cover photo** — can be set now or added later, once you've uploaded photos
- **Privacy** — Private or Link-only (see [Public vs private galleries](./3-3-gallery-visibility.md))

Once created, the gallery generates its own unique encryption key automatically, behind the scenes — this is what makes every gallery's content independently protected. You don't need to do anything for this step; it happens the moment you create the gallery.

After creation, you'll land in the empty gallery, ready to upload. See [Uploading photos to a gallery](./3-2-upload-photos.md).
